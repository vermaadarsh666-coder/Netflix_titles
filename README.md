Netflix Shows Analysis Dashboard

This project is a data analysis dashboard created using Tableau. It analyzes Netflix shows and presents insights through different visualizations.

Introduction

The purpose of this project is to explore Netflix shows data and understand patterns in the content available on the platform. The dashboard helps analyze how Netflix shows are distributed across different countries, directors, cast members, and categories.

Dataset

The dataset contains information about Netflix titles including show ID, title, director, cast, country, category, type of show, and release year.

Tech Stack

Tableau was used to build the interactive dashboard.
Microsoft Excel was used for basic data cleaning and preparation.

Features

Shows distribution by country using a world map
Number of shows created by different directors
Shows by cast members
Shows by category or genre

Insights

The dashboard shows that some categories like drama and comedy have the highest number of shows. It also highlights that Netflix content is produced in many different countries and involves a wide range of directors and actors.
