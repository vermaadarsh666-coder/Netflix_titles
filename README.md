Netflix Shows Analysis Dashboard

This project is a data analysis dashboard that explores Netflix shows using visualizations. It helps understand how Netflix content is distributed across different countries, directors, cast members, and categories.

Introduction

The purpose of this project is to analyze Netflix shows and present insights through a dashboard. The dashboard makes it easy to explore patterns in Netflix content such as popular genres, global distribution of shows, and contributions of different directors and actors.

Dataset

The dataset contains information about Netflix titles including details like show ID, title, director, cast, country, category, type of show, and release year.

Tech Stack

Power BI was used to create the interactive dashboard and Microsoft Excel was used for basic data cleaning and preparation.

Features

* Shows distribution by country using a world map
* Number of shows created by different directors
* Shows by cast members
* Shows by category or genre

Insights

The dashboard shows that some categories like drama and comedy have the highest number of shows. It also highlights that Netflix content is produced in many different countries and involves a wide range of directors and actors.
